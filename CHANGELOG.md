## [1.0.0] - 2026-07-05

### Added

- Initial release of Domination.
- 250+ modern-era countries with neighbour adjacency, AI diplomacy (alliance, NAP, embargo, tribute), economy, happiness, and conquest wars.
- Land invasions and sea-borne invasions with distinct bleed/refund curves.
- 3D globe rendering: day/night/specular Earth, FBM-driven clouds, atmosphere halo, soft shadows.
- Tauri 2 desktop app (Windows NSIS + Linux AppImage) with auto-update from GitHub Releases.
- Stable + opt-in beta update channels, configurable in Settings.
- 24h throttled auto-check for updates; manual "Check for updates" in Settings.
- Welcome changelog shown the first time the new version launches.
